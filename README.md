Java Swing ile Modern Hesap Makinesi Arayüzü
Bu proje, Java programlama dili ve Swing kütüphanesi kullanılarak geliştirilmiş modern bir Hesap Makinesi masaüstü arayüz simülasyonudur.

[!IMPORTANT]
Proje Durumu: Bu çalışma şu aşamada ağırlıklı olarak Grafiksel Kullanıcı Arayüzü (GUI) tasarımına odaklanmıştır. Bileşenlerin yerleşimi ve görsel hiyerarşi üzerine pratik yapmak amacıyla geliştirilmiştir.

🚀 Proje Özellikleri
Bileşen Yönetimi: JFrame, JPanel, JButton ve JTextField gibi temel Swing yapıları kullanılarak temiz bir görünüm elde edilmiştir.

Geniş Fonksiyon Paneli: Standart dört işlemin yanı sıra yüzde (%), kare alma, karekök ve 1/x gibi gelişmiş fonksiyon butonları tasarıma dahil edilmiştir.

Modern Tema: Kullanıcı deneyimini iyileştirmek adına koyu tema tercihi ile modern ve şık bir arayüz oluşturulmuştur.

🛠️ Teknik Detaylar ve Geliştirme Ortamı
Proje, aşağıdaki araçlar ve ortam kullanılarak hayata geçirilmiştir:

İşletim Sistemi: Windows 11

IDE: Apache NetBeans IDE 20

Dil: Java

Kütüphaneler: Java Swing & AWT

💻 Kurulum ve Çalıştırma
Bu depoyu (repository) yerel makinenize klonlayın.

Apache NetBeans editörünü açın.

File -> Open Project kısmından proje klasörünü seçin.

Run (F6) butonuna basarak arayüzü görüntüleyebilirsiniz.
